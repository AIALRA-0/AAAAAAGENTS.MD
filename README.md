# AAAAAAGENTS.MD

This repository is split into two fully separated workspaces with the same structure:

- [AAAAAAGENTS.MD.CN](./AAAAAAGENTS.MD.CN/) for Chinese constraints, scripts, web UI, and docs
- [AAAAAAGENTS.MD.EN](./AAAAAAGENTS.MD.EN/) as a 1:1 English translation of the CN workspace

## Description

A dual-language AGENTS governance workspace that combines:

- constraint-driven execution (`AGENTS.md`, `MILESTONE.md`, `CHANGE.md`, `TREE.md`)
- deterministic toolchain (`agents_tools`)
- local visualization dashboard (`agents_web`)
