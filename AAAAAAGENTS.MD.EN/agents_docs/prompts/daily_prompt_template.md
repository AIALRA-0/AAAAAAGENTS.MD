# Daily Conversation Prompt Template

Use this template for daily task requests in an already initialized project.

## English

### 1) Today's Goal

- Today's target: `<TODAY_GOAL>`
- Priority level: `<PRIORITY>`
- Deadline or expected completion: `<TARGET_TIME>`

### 2) Current Status

- Current milestone/node: `<CURRENT_MILESTONE>`
- Current blockers: `<CURRENT_BLOCKERS>`
- Latest known state: `<CURRENT_STATE>`

### 3) Requested Action

- Requested task: `<REQUESTED_ACTION>`
- Files expected to change: `<EXPECTED_FILES>`
- Files that must not change: `<PROTECTED_FILES>`

### 4) Risk / Uncertainty

- Known risk: `<RISK_ITEM>`
- Unknowns needing confirmation: `<OPEN_QUESTIONS>`
- Fallback strategy: `<FALLBACK_PLAN>`

### 5) Expected Output Format

- Update summary style: `<SUMMARY_FORMAT>`
- Required command outputs: `<REQUIRED_OUTPUTS>`
- Required verification command: `<VERIFY_COMMAND>`

### 6) End-of-Day Summary

- What was done: `<DONE_ITEMS>`
- What failed or was deferred: `<NOT_DONE_ITEMS>`
- Suggested next step: `<NEXT_STEP>`
