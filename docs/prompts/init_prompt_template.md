# Initialization Prompt Template

Use this template when bootstrapping a new project with the AGENTS.md workflow.

## English

### 1) Context Collection

- Project name: `<PROJECT_NAME>`
- Domain or product type: `<PROJECT_DOMAIN>`
- Existing files/resources: `<EXISTING_CONTEXT>`
- Constraints to follow: `<CONSTRAINT_FILES>`

### 2) Task Objective

- Primary goal: `<TASK_GOAL>`
- Deliverables: `<DELIVERABLE_LIST>`
- Out of scope: `<OUT_OF_SCOPE>`

### 3) Constraints

- Editable scope: `<EDITABLE_SCOPE>`
- Read-only scope: `<READ_ONLY_SCOPE>`
- Validation rules: `<VALIDATION_RULES>`
- Timestamp format: `<TIME_FORMAT>`

### 4) Output Contract

- Expected output format: `<OUTPUT_FORMAT>`
- Required updated files: `<REQUIRED_FILES>`
- Required commands to run: `<REQUIRED_COMMANDS>`
- Verification evidence: `<VERIFICATION_EVIDENCE>`

### 5) Acceptance Checklist

- [ ] Core files are created/updated as requested
- [ ] File structure is aligned with constraints
- [ ] Validation commands return success
- [ ] Changes are documented in milestone/change/tree flow
- [ ] No out-of-scope files were modified

## 中文

### 1) 上下文收集

- 项目名称：`<PROJECT_NAME>`
- 业务/产品类型：`<PROJECT_DOMAIN>`
- 现有文件与资源：`<EXISTING_CONTEXT>`
- 需遵循约束：`<CONSTRAINT_FILES>`

### 2) 任务目标

- 核心目标：`<TASK_GOAL>`
- 预期交付：`<DELIVERABLE_LIST>`
- 明确不做：`<OUT_OF_SCOPE>`

### 3) 约束条件

- 可编辑范围：`<EDITABLE_SCOPE>`
- 只读范围：`<READ_ONLY_SCOPE>`
- 校验规则：`<VALIDATION_RULES>`
- 时间戳格式：`<TIME_FORMAT>`

### 4) 输出契约

- 输出格式要求：`<OUTPUT_FORMAT>`
- 必须更新文件：`<REQUIRED_FILES>`
- 必须执行命令：`<REQUIRED_COMMANDS>`
- 验证依据：`<VERIFICATION_EVIDENCE>`

### 5) 验收清单

- [ ] 核心文件按要求创建或更新
- [ ] 文件结构与约束一致
- [ ] 校验命令执行成功
- [ ] 里程碑/变更/树记录链路完整
- [ ] 未修改越界文件
