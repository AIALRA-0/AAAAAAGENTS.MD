# Screenshot Capture Guide

This folder stores screenshot assets referenced by `README.md`.

## Expected Files

- `01-overview.png`
- `02-milestone-flow.png`
- `03-tree-explorer.png`
- `04-edit-mode.png`

These files are placeholders by contract at bootstrap stage and are not generated automatically.

## Capture Targets

1. `01-overview.png`
- Capture the default landing view after opening the local dashboard.
- Include top toolbar, left navigation, and main content area.

2. `02-milestone-flow.png`
- Open `MILESTONE.md` in graph mode.
- Capture the full flow graph and node detail panel.

3. `03-tree-explorer.png`
- Open `TREE.md` explorer view.
- Capture the tree navigation and selected node detail.

4. `04-edit-mode.png`
- Enter edit mode on a structured document.
- Capture raw editor + node editor area together.

## Recommended Specs

- Resolution: `1920x1080` (desktop) or `1366x768` minimum
- Aspect ratio: `16:9`
- Format: `.png`
- UI language: either English or Chinese, but keep one language per screenshot set

## Naming Rules

- Keep exact filenames above.
- Use lowercase and dash-separated names.
- Replace files in place to avoid broken README links.

## Replacement Steps

1. Capture the target screen.
2. Export as PNG.
3. Rename to the exact expected filename.
4. Put it in this folder.
5. Reopen `README.md` and verify all screenshot links.
